Probabilistic Bias Analysis for an Unmeasured Confounder

Description

Monte Carlo version of deterministic_bias_analysis(): instead of a single set of assumptions, the bias parameters are drawn repeatedly from prior distributions and combined with the sampling uncertainty in the observed risk difference, yielding a simulation interval for the bias-adjusted effect. This ports the probabilistic unmeasured-confounding sensitivity analysis of García-Albéniz et al. (supplementary analysis).

Usage

probabilistic_bias_analysis(
  rd,
  prev_continue,
  prev_stopbase,
  confounder_effect,
  rd_se = 0,
  n_sim = 1000L,
  conf_level = 0.95,
  seed = NULL
)

Arguments

rd Observed arm risk difference (CONTINUE minus STOPBASE).
prev_continue, prev_stopbase, confounder_effect Either a single value (fixed) or a length-2 c(min, max) uniform range, as described above.
rd_se Standard error of the observed risk difference. Default: 0 (ignore sampling uncertainty, propagate only the bias parameters).
n_sim Number of Monte Carlo iterations. Default: 1000L.
conf_level Width of the reported simulation interval. Default: 0.95.
seed Optional integer seed. The caller’s RNG state is restored on exit. Default: NULL.

Details

On each of n_sim iterations the function draws the observed risk difference from a normal distribution (rd, rd_se), draws each bias parameter, and computes the bias-adjusted risk difference rd_draw - (prev_continue - prev_stopbase) * confounder_effect. The reported interval is the central conf_level percentile range of those draws.

Each bias parameter (prev_continue, prev_stopbase, confounder_effect) may be given as a single number (held fixed) or as a length-2 vector c(min, max) (drawn from a uniform distribution on that range). Set rd_se to 0 to propagate only the bias-parameter uncertainty.

Value

A named list with estimate (median adjusted risk difference), conf_low and conf_high (the simulation-interval bounds), draws (the n_sim adjusted risk differences), n_sim, and conf_level.

References

García-Albéniz X, Uno H, Bhatt DL, McArdle PH, Joffe MM, Hernán MA. Continuation of Annual Screening Mammography and Breast Cancer Mortality in Women Older Than 70 Years: A Prospective Observational Study. Ann Intern Med. 2020;172(6):381-389. doi:10.7326/M18-1199

See Also

deterministic_bias_analysis() for the single-assumption version.
